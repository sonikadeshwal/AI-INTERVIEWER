🤖 AI Interviewer

An AI-powered mock interview platform that simulates real interview conversations and helps users practice technical or behavioral interviews interactively.

The application generates role-based interview questions using AI and conducts a conversational interview session to help candidates improve their interview skills.

🧠 How It Works

User opens the web application

User selects or enters an interview role

AI generates relevant interview questions

User answers the questions

AI continues the interview conversation dynamically

Run the application
